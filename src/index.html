Click the card pile (face down) to draw a new random card!


Simple on the outside, complicated on the inside! This is my attempt at making a random card genrator the hard way, using DOM manipulation, if statements, the forEach method, and event listeners.


A simpler method would use HTML onclick, and CSS :hover for the mouseover. However, I followed the lesson as taught for a fun challenge. In upcoming projects, I will use React and make things a lot easier!
